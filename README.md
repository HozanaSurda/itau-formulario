# Itaú - Desafio Front-end - Hozana Raquel

## O objetivo foi desenvolver um formulário de cadastro com os seguintes campos:

-  Nome 
-  Cpf
-  Rg
-  Sexo
-  Endereço
-  Número
-  Bairro
-  Cidade
-  Estado
-  Cep
-  Telefone Fixo
-  Telefone Celular

## Como regra de negócio, foi exigido que os seguintes campos fossem obrigatórios:
-  Nome 
-  Cpf
-  Endereço
-  Número
-  Telefone Celular
